1. Install tailwind - "npx tailwindcss -o tailwind.css"
2. Link tailwind.css to index.html
3. Create custom style.css file in /src
4. Add tailwind utilities (@tailwind base; @tailwind components; @tailwind utilities;)
5. Add build script "npx tailwindcss -i ./src/styles.css -o ./dist/tailwind.css" in package.json
6. Create tailwind config file - "npx tailwindcss init"
